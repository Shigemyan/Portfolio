# ポートフォリオ

## 柴田琴美

メール・データ・繰り返し作業を整え、日々の業務が滞りなく進む仕組みづくりを支援します。

製本会社で約15年、機械オペレーターと品質チェックを担当してきました。  
現場で培った「作業の流れを整理する力」と「正確に続けられる形に整える視点」を活かし、在宅での事務支援・業務改善・小さな自動化に取り組んでいます。

## お手伝いできること

- メール・Googleスプレッドシートなど、日常業務の整理
- データ入力、集計、進捗管理、定型作業の見直し
- 手順書・業務マニュアルの作成、更新
- 定型メールや問い合わせ対応の下書き作成
- 繰り返し作業を減らすための、Excel・Google Workspaceを使った仕組みづくり
- 小さな自動化の提案・試作・運用しやすい形への改善

「どこから整理したらよいか分からない」「手作業が増えて本来の仕事に集中できない」といったご相談から、一緒に状況を整理します。

## 強み

### 現場を理解して整理する

作業の流れを見て、困っていること・手間がかかること・間違いが起きやすい場所を言葉にします。  
いきなり大きな仕組みを作るのではなく、まずは業務に合う小さな改善から始めます。

### 正確に続けられる形に整える

品質チェック業務の経験を活かし、使う人が迷いにくく、引き継ぎやすい形を大切にしています。  
ツールを作ること自体ではなく、毎日の仕事が少し楽になることを目標にします。

### 必要に応じて自動化まで対応する

Excel、Googleスプレッドシート、Python、Google Apps Script、Difyなどを活用し、定型作業の削減や情報整理を支援します。

## 制作・改善の例

### パレット看板自動作成ツール

**課題**  
大量の看板を手作業で作成しており、作成に時間がかかっていました。

**対応**  
Excel VBAを用いて、パレット看板を自動作成する仕組みを作りました。

**変化**  
作成作業を短縮し、作業負担の軽減と業務の効率化につなげました。

### AI秘書

**課題**  
メールで流れてしまう情報を、あとから探せる形で残したいと考えました。

**対応**  
Gmailの情報をGoogleスプレッドシートへ整理し、質問形式で検索・再利用できる仕組みを試作しました。

**使用技術**  
Google Apps Script / Gmail / Google Sheets / Dify / OpenAI API

> 公開している画面・データは、個人情報や機密情報を含まない検証用のものです。

### 受注集計・在庫更新・メール通知の自動化

受注データの集計、在庫更新、メール通知までの定型作業を、Pythonでまとめて処理するプログラムを制作しました。

## ご依頼からの進め方

1. 現在の作業と困りごとを伺います
2. 優先順位と、まず試す小さな改善をご提案します
3. 実際に使える形を作り、運用に合わせて調整します
4. 必要に応じて、使い方や手順をまとめてお渡しします
